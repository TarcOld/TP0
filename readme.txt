Inicialmente, intenté hacer una sola clase de Elemento. El 
problema de esto era que traía consigo el uso de muchos if y de
métodos de varias líneas. Luego se me ocurrió la idea de
utilizar una interfaz también para el elemento. De esta forma
logré cumplir en casi todos los métodos el requerimiento de una
sola línea de código y no usar if. El "Add" de ElementoLleno
quedó con 3 líneas y un if. Luego se me ocurrió cómo hacerlo en
2 líneas y sin if. Pero no logré pensar cómo hacerlo en menos de
esa cantidad de líneas. Lo mismo sucede con un constructor de
la misma clase.
